## 章構成の方針

- 1ファイル=1章。ファイル名は `NN-英語スラッグ.md`(章番号ゼロ埋め2桁)。
- 章をまたぐ参照は相対リンクで張る(例:第2章から第1章の節へ)。
- 各章の末尾に「確認事項/確定事項」を置き、レビューの状態を可視化する。
